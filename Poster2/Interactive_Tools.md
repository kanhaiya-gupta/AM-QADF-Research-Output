# Interactive Tools

## Seven Interactive Jupyter Notebooks

### 1. Data Warehouse Explorer
- **Purpose**: Explore warehouse data
- **Features**: Query interface, data visualization, filtering
- **Users**: All users

### 2. Signal Mapping Interface
- **Purpose**: Create and configure voxel grids
- **Features**: Grid type selection, resolution settings, signal mapping
- **Users**: Process engineers, data analysts

### 3. Voxel Visualizer
- **Purpose**: 3D visualization of voxel grids
- **Features**: Interactive 3D rendering, signal selection, slicing
- **Users**: All users

### 4. Quality Analytics Dashboard
- **Purpose**: Quality assessment and metrics
- **Features**: Quality metrics, completeness analysis, signal quality
- **Users**: Quality engineers, process engineers

### 5. Fusion and Anomaly Detection
- **Purpose**: Multi-signal fusion and anomaly detection
- **Features**: Fusion strategies, anomaly detection methods, visualization
- **Users**: Data analysts, quality engineers

### 6. Advanced Analysis
- **Purpose**: Sensitivity analysis and virtual experiments
- **Features**: 12 sensitivity methods, 10 experiment designs, results visualization
- **Users**: Data analysts, researchers

### 7. End-to-End Workflow
- **Purpose**: Complete analysis workflow
- **Features**: Guided workflow, all capabilities integrated
- **Users**: All users
