## Design Guidelines

### Layout
- **Title**: Top, large and bold (include LuFo VII branding)
- **Introduction**: Top-left (research context)
- **Use Cases**: Top-right (three scenarios)
- **Results**: Middle-left (quantitative results)
- **Business Value**: Middle-right (cost savings, ROI)
- **Lessons Learned**: Bottom-left
- **Conclusion**: Bottom-right

### Color Scheme
- **Background**: White or light gray
- **Primary Text**: Dark gray or black
- **Accents**: Use color to distinguish use cases
- **LuFo VII Branding**: Include program colors/logos if available
- **Consistency**: Same color scheme across all figures

### Typography
- **Title**: 72-96 pt, bold
- **Section Headings**: 36-48 pt, bold
- **Body Text**: 18-24 pt
- **Figure Captions**: 14-16 pt
- **Font**: Sans-serif (Arial, Helvetica)

### Figure Requirements
- **Resolution**: 300 DPI minimum
- **Format**: High-quality PNG or vector graphics
- **Labels**: All axes, legends clearly labeled
- **Readability**: Text readable from 1-2 meters
- **Industry Context**: Include aerospace/aircraft component imagery where appropriate
